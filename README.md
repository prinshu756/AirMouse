 AIRMouse – Control Your Mouse with Hand Gestures

AIRMouse is a Python-based virtual mouse that lets you control your computer cursor using hand gestures captured through a webcam.  
It uses MediaPipe for real-time hand tracking and PyAutoGUI for controlling the cursor, clicks, and scrolls — all without touching a mouse!

 Features
- Move Cursor: Move your index finger to control the cursor position.  
- Click: Open all five fingers to perform a click.  
- Scroll:  
  - Two fingers up → Scroll down  (speed may changes as according of your system)
  - Three fingers up → Scroll up  
- Smooth Tracking: Exponential smoothing for stable and accurate motion.  
- Real-time Hand Detection: Uses OpenCV and MediaPipe for fast and accurate hand tracking.

---

Requirements

Make sure you have Python 3.8+ installed.

Install the required libraries:

Numpy , openCV , Mediapipe and Pyautogui
